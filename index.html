WHAT IS WARTHOG?

Warthog is an experimental cryptocurrency with world's first Proof of Balanced Work algorithm written by a group of crypto enthusiasts. This project is goes to be serious and everyone is welcome to join and contribute. Who knows how big this project will be?

TOKENOMICS

This is not the default kind of project where you see a disproportionately high allocation of the total supply unfairly reserved for the team, the advisors and their pockets. 0% is minted, premined, reserved for anyone, 100% is publicly mineable.

Аfter launch, node will start synchronizing the blockchain, after it is completed, it will be possible to perform operations with the wallet and start mining.

WALLET

To use a wallet, you need to download pre-compiled wallet executable file from here. Currently support Windows, macOS (only arm), Linux (Ubuntu, Debian), but you can compile

Create wallet

To create the wallet you need just run next command: 
./wart-wallet -c
In output you get wallet address, private and public keys. This information will be saved to the wallet.json file.

Check balance

If you want check your balance run command: 
./wart-wallet -b
In output you get balance of your wallet

Send coins

To send some coins, use command: 
./wart-walet -s --to=%address% --amount %count% --fee 0
./wart-walet -s --to=9da0dc31d0c3bcaaeaf422be6499fbf451b314fa622be978 --amount 0.00000001 --fee 0
Next, you will be shown the details of the transaction and if you agree with them, you need to confirm it

To restore your wallet.json file, you need run command: 
./wart-wallet -r %your private key%
In output you get wallet address, private and public keys and

MINER

To start mine this coin you need to download janusminer here. Current stable version is 0.0.4. Now miner have ONLY Linux support with OpenCL API. 
./janusminer -a %wallet% -h %host_ip% -p %node_port%
